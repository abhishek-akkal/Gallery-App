# 🌌 Gallery App

A responsive Gallery App built using **React**, **Axios**, and **Tailwind CSS** with a modern **space-themed UI**.

Users can explore images, move between pages, and enjoy interactive hover effects with a glassmorphism inspired design.

## ✨ Features

- 📷 Image fetching using API
- 🔄 Pagination (Prev / Next)
- ⚡ Loading state
- 🎨 Space themed UI
- 🌌 Glow effects & animations
- 🪟 Glassmorphism design
- 📱 Responsive layout
- 🖱 Interactive hover effects
- ⚛ Component based React structure
